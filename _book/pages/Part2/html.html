HTML代码书写规范

HTML代码书写规范

1.根据不同的页面功能，body下的第一个元素必须添加声明class，实例如下：

<div class="page-listHtml"> // 列表页
<div class="page-addtHtml"> // 添加页
<div class="page-viewHtml"> // 查看页

2.所有页面的按钮、组件的class必须一致，每个按钮都有例如常用的增删改查：

<div class="page-listHtml"> // 列表页
<div class="page-addtHtml"> // 添加页
<div class="page-viewHtml"> // 查看页
